# tic tac toe vs computer (c++)

console game where you play tic tac toe against the computer

# what is this
i made this project to practice c++ basics like arrays, loops, functions and simple ai logic.
the game lets you play X while the computer plays O.

# features
- 3x3 grid display
- human vs computer
- basic ai for computer moves (blocks you and tries to win)
- input validation
- detects win, draw, and invalid moves

# how to run
run the program using your c++ compiler or IDE (like codeblocks or vs code)

# example
---Tic Tac Toe vs computer---
You are X , AI is O

Pick a number (1-9): 5
[grid updates]

you win! / computer wins! / draw!

# limits
- no difficulty levels
- only 1 player vs computer
